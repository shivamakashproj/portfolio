# 🌐 Personal Portfolio Website

A responsive and modern personal portfolio website developed using **HTML**, **CSS**, and **JavaScript**. This project showcases my skills, projects, education, certifications, and contact information. It is hosted using **AWS S3 Static Website Hosting**.

This portfolio website serves as my online professional profile where visitors can learn about me, explore my projects, and view my technical skills.

## ☁️ AWS Static Website Hosting

This website is deployed using **Amazon S3 Static Website Hosting**.

### Deployment Steps

1. Created an S3 Bucket.
2. Uploaded all website files.
3. Enabled Static Website Hosting.
4. Configured Bucket Policy for public access.
5. Accessed the website through the S3 Website Endpoint.
